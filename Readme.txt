Proyecto - "0Landing"
Por AlternFourty

Como aprendiz de FrontEnd Development presento el siguiente como mi proyecto principal para aplicar todo
el contenido que he aprendido durante la carrera profesional de FrontEnd en http://platzi.com -

El proyecto se conforma basicamente del desarrollo de una landing page para mi marca personal, con un 
diseño propio y llevado a codigo con bootstrap. La landing sera dividida en secciones:

- Index, donde se encontrara info basica como mi nickname y una bio desplegable. ademas de Anchors para 
las demas secciones y sitios externos que deseo anexar.

- Portafolio, Aqui sera donde empezare a anexar cada uno de los proyectos en los que comienze a hacer
parte conforme crezca en la industria.

- Art and games, En esta seccion incluire accesos a mis perfiles de pixiv y algunos detalles sobre mi 
progreso en el mundo gamer (y por que no, si decido desarrollar un juego tambien estaria aqui)

- Social Networks

- Anime Sector, En esta seccion me gustaria compartir un pequeño resumen de las series anime y manga que
me encuentro viendo/leyendo.

Mi intencion es que este documento HTML sea unico, es decir no existirian otros documentos para el resto
de secciones (social.html, portafolio.html, etc.) unicamente index.html. es decir me valdria del poder de
JavaScript para lograr que mi Landing page sea una especie de Slider. lo interesante de este proyecto es
que actualmente. (es decir el dia que escribo esto) mi conocimiento de JavaScript es totalmente basico.
asi que este proyecto tambien tiene una funcion de automotivador. por el momento esta seria la secuencia
de desarrollo.

- Documentacion Basica
- Documento HTML/CSS inicial
- Integracion de Bootstrap
- Crear estructura basica de la seccion Index
- Crear o implementar elementos visuales externos
- Implementar Funcionalidad slider con JS
